================================================================================
                    JT FENCE INC. - QUICKSTART GUIDE
================================================================================

Welcome! Your complete Next.js website is ready. Follow these 3 simple steps:

================================================================================
STEP 1: INSTALL DEPENDENCIES (2 minutes)
================================================================================

Open your terminal and run:

    cd /sessions/nifty-jolly-ride/jt-fence
    npm install

This will install all required packages (Next.js, React, Tailwind, etc.)

================================================================================
STEP 2: START DEVELOPMENT SERVER (1 minute)
================================================================================

Still in the same directory, run:

    npm run dev

You should see:
    ✓ ready - started server on 0.0.0.0:3000

Open your browser to: http://localhost:3000

You'll see the complete JT Fence Inc. website with:
  - Professional home page with hero section
  - Services page
  - About page with team information
  - Portfolio/gallery page
  - Contact form
  - Footer with links

================================================================================
STEP 3: CUSTOMIZE FOR YOUR BUSINESS (5-10 minutes)
================================================================================

Update these files with your actual information:

A. Company Information
   File: .env.local
   - Update NEXT_PUBLIC_PHONE
   - Update NEXT_PUBLIC_EMAIL

   File: src/components/Header.tsx (line 11)
   - Update phone number if needed

   File: src/components/Footer.tsx (lines 4-7)
   - Update phone, email, address

B. Services & Portfolio
   File: src/data/services.ts
   - Add your fence services

   File: src/data/portfolio.ts
   - Add your completed projects

C. Team & About
   File: src/app/about/page.tsx
   - Update your mission, values, team info

D. Contact & Hours
   File: src/app/contact/page.tsx
   - Update hours, service areas, address

================================================================================
USEFUL COMMANDS
